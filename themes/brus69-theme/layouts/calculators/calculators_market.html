Онлайн калькуляторы для маркетинга

Выберите нужный инструмент и начните расчеты прямо сейчас

CAC

(Стоимость привлечения клиента)

Расчет производится по формуле CAC = общие затраты на маркетинг / количество новых клиентов

РАССЧИТАТЬ

CPM

(Стоимость за 1000 показов)

Расчет производится по формуле CPM = (общие расходы на рекламу / количество показов) * 1000

РАССЧИТАТЬ

LTV

(Пожизненная ценность клиента)

Расчет производится по формуле LTV = средний чек * частота покупок * средняя продолжительность отношений с клиентом

РАССЧИТАТЬ

ROI

(Возврат на инвестиции)

Расчет производится по формуле ROI = (доход - расходы) / расходы * 100%

РАССЧИТАТЬ

CPA

(Стоимость за действие)

Расчет производится по формуле CPA = общие расходы на рекламу / количество целевых действий

РАССЧИТАТЬ

CPL

(Привлечение одного лида с сайта)

Расчет производится по формуле CPL = расход на рекламу / количество лидов

РАССЧИТАТЬ

CR

(Коэффициент конверсии)

Расчет производится по формуле CR = (количество целевых действий / количество посетителей) * 100%

РАССЧИТАТЬ

CPC

(Цена клика)

Расчет производится по формуле CPC = потраченная сумма / количество кликов

РАССЧИТАТЬ

ROAS

(Рентабельность затрат на рекламу)

Расчет производится по формуле ROAS = (доход от рекламы / расходы на рекламу) * 100%

РАССЧИТАТЬ

ROMI

(Коэффициент возврата маркетинговых инвестиций)

Расчет производится по формуле ROMI = (доход-расходы) / расходы * 100%

РАССЧИТАТЬ

DRR

(Доля рекламных расходов)

Расчет производится по формуле DRR = (затраты на рекламу / прибыль с рекламы) * 100%

РАССЧИТАТЬ

Прибыльность

(Общая рентабельность)

Расчет производится по формуле Прибыльность = (прибыль / выручка) * 100%

РАССЧИТАТЬ

Бюджет

(Маркетинговый бюджет)

Расчет производится по формуле Бюджет = ожидаемая выручка * процент от выручки на маркетинг

РАССЧИТАТЬ

Удержание

(Коэффициент удержания)

Расчет производится по формуле Удержание = (количество удержанных клиентов / общее количество клиентов) * 100%

РАССЧИТАТЬ

Отток

(Коэффициент оттока)

Расчет производится по формуле Отток = (количество потерянных клиентов / общее количество клиентов) * 100%

РАССЧИТАТЬ

EBITDA

(Прибыль до вычета процентов, налогов и амортизации)

Расчет производится по формуле EBITDA = прибыль + проценты + налоги + амортизация

РАССЧИТАТЬ

MRR/ARR

(Ежемесячный/годовой повторяющийся доход)

Расчет производится по формуле MRR = количество клиентов * средний месячный доход с клиента

РАССЧИТАТЬ

Когорты

(Анализ когорт)

Расчет производится по формуле Когорта = группа пользователей, объединенных по дате первого действия

РАССЧИТАТЬ

Экономика единицы

(Экономика единицы продукта)

Расчет производится по формуле Экономика единицы = доход с единицы - стоимость единицы

РАССЧИТАТЬ

Северная звезда

(Ключевой показатель роста)

Расчет производится по формуле Северная звезда = ключевой показатель, который отражает ценность продукта для пользователя

РАССЧИТАТЬ

NPS

(Индекс лояльности клиентов)

Расчет производится по формуле NPS = % промоутеров - % детракторов

РАССЧИТАТЬ

Размер выборки

(Необходимый размер выборки)

Расчет производится по формуле Размер выборки = Z² * p * (1-p) / e²

РАССЧИТАТЬ

АВ-тесты

(Калькулятор АВ-тестов)

Расчет производится по формуле АВ-тест = сравнение двух версий страницы для определения более эффективной
